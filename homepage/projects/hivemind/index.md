HiveMind
=====

A Magic: The Gathering draft tool

## Download

[hivemind-0.1.0.tar.gz](/blob/hivemind-0.1.0.tar.gz)

## Dependencies

- Boost

## Build and Run

```
cd hivemind
make && ./hivemind
```
